# Contributing to NeckGuard

感谢你对 NeckGuard 的关注！以下是参与贡献的指南。

## 🚀 快速开始

1. Fork 本仓库
2. Clone 到本地：`git clone https://github.com/your-username/neck-guard.git`
3. 安装依赖：`npm install`
4. 启动开发：`npm run dev`

## 📝 如何贡献

### 修复 Bug

1. 在 Issues 中搜索是否已有相关 Bug
2. 如果没有，创建新 Issue 描述问题
3. Fork 仓库，创建修复分支：`git checkout -b fix/your-bug-fix`
4. 提交修复并创建 Pull Request

### 添加功能

1. 先创建 Issue 讨论你的想法
2. 等待维护者反馈
3. Fork 并开发：`git checkout -b feature/your-feature`
4. 提交 PR

### 添加知识库内容

在 `src/data/knowledge.js` 中添加文章，格式参考现有文章。

### 添加康复运动

在 `src/data/exercises.js` 中添加运动，格式参考现有运动。

## 📋 代码规范

- 使用 2 空格缩进
- 组件使用函数式组件 + Hooks
- 命名：组件 PascalCase，其他 camelCase
- 提交信息使用中文或英文均可

## 🧪 测试

```bash
npm run build  # 确保构建成功
```

## 📄 Pull Request 规范

- 标题清晰描述修改内容
- 如果修改了 UI，请附上截图
- 确保所有代码能正常构建

## ❤️ 社区准则

- 尊重所有贡献者
- 建设性的反馈和讨论
- 关注问题本身，而非个人
