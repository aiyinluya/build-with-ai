# 快速诊断指南

## 如果看到骨骼图报错，请按以下步骤排查：

### 1. 检查浏览器控制台
- 按 F12 打开开发者工具
- 查看 Console 标签页
- 记录具体的错误信息

### 2. 常见错误及解决方案

**错误1: "Cannot read property 'category' of undefined"**
- 原因: exercise 对象为空
- 解决: 检查运动数据是否正确加载

**错误2: "SVG 元素不显示"**
- 原因: SVG 属性问题
- 解决: 已修复，重新加载页面

**错误3: "组件未导入"**
- 原因: 导入路径错误
- 解决: 检查 ExercisePage.jsx 中的导入语句

### 3. 快速修复步骤

```bash
# 1. 清除缓存
rm -r node_modules/.vite

# 2. 重启开发服务器
npm run dev

# 3. 硬刷新浏览器
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)
```

### 4. 如果还是不行

请告诉我：
1. 浏览器控制台的具体错误信息
2. 你点击的是哪个运动
3. 页面的其他部分是否正常显示
