# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-24

### Added
- 症状自测问卷 (F001) - 18道专业问题评估颈椎健康
- 健康评分系统 (F002) - 基于问卷的智能评分
- 康复方案推荐 (F003) - 4种预制训练方案
- 运动库 (F004) - 23个专业康复运动
- 每日追踪记录 (F005) - 症状和运动记录
- 趋势图表 (F006) - Chart.js数据可视化
- 知识库 (F007) - 10篇专业科普文章
- 健康报告 (F008) - 周报/月报生成
- 暗色模式 (F009) - 主题切换
- 提醒设置 (F010) - 浏览器通知
- PWA支持 (F011) - 可添加到桌面
- 数据导出 (F012) - CSV和Markdown格式
- 成就系统 (F013) - 10个成就徽章
- 个人中心 (F014) - 用户数据统计
- 每日提示 (F015) - 随机健康小贴士
- 运动动画 (F016) - SVG动画演示
- B站视频 (F017) - 视频教程嵌入

### Technical
- React 18 + Vite 5
- TailwindCSS 样式系统
- Chart.js 图表可视化
- React Router 6 路由
- LocalStorage 数据持久化
- PWA Manifest
