# NeckGuard 功能更新 - 2026-03-26

## 🎉 新增功能

### 1. 智能推荐系统
**文件：** `src/utils/recommendations.js` + `src/components/SmartRecommendations.jsx`

**功能：**
- 根据诊断结果自动推荐康复方案
- 推荐针对性运动
- 推荐相关知识文章
- 生成个性化健康建议
- 严重症状警告提示

**集成位置：** 首页 "为你推荐" 模块

---

### 2. 新手引导
**文件：** `src/components/OnboardingGuide.jsx`

**功能：**
- 首次访问自动弹出
- 4屏引导介绍核心功能
- 进度条和跳过按钮
- localStorage 记录状态

**内容：**
1. 欢迎页
2. 症状自测介绍
3. 康复运动介绍
4. 追踪进度介绍

---

### 3. 分享打卡
**文件：** `src/components/ShareCard.jsx`

**功能：**
- 运动完成后生成打卡卡片
- 显示运动信息和打卡统计
- 一键复制分享文案
- 支持分享到社交媒体

**集成位置：** ExerciseTimer 完成后

---

### 4. 骨骼动画系统
**文件：** `src/data/skeletonAnimations.js` + `src/components/SkeletonPlayer.jsx`

**功能：**
- 5个核心动作的骨骼动画
- 帧插值平滑过渡
- 高亮关键肌肉部位
- 箭头标注运动方向
- 可调节播放速度
- 点击进度条跳转

**已实现动画：**
1. 收下巴 (chin_tuck)
2. 上斜方肌拉伸 (trap_stretch)
3. 肩胛骨后缩 (scapular_retraction)
4. 颈部侧向拉伸 (neck_side_stretch)
5. 肩部耸肩放松 (shoulder_shrug)
