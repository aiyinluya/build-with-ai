## 概述

基于症状自测问卷的答案，计算用户的健康评分和风险等级，并提供个性化建议。

## 设计

### 评分算法

```javascript
// 计算各维度得分
scores.symptoms = 颈椎 + 肩膀 + 背部 + 手臂 + 头痛
scores.work = 工作时长 + 休息频率 + 显示器 + 姿势 + 压力
scores.sleep = 睡姿 + 枕头 + 保暖 + 睡眠质量
scores.history = 持续时间 + 运动频率 + 既往治疗

// 总分计算
total = sum(scores)
maxPossible = 72  // 18题 × 4分
percentage = (total / maxPossible) × 100

// 风险等级判定
if (percentage <= 20) → 低风险
if (percentage <= 45) → 中等风险
if (percentage <= 70) → 较高风险
if (percentage > 70) → 高风险
```

### 问题区域识别

根据各维度得分，识别需要重点关注的问题：
- 症状得分 > 10 → 症状明显
- 工作得分 > 10 → 工作习惯需改善
- 睡眠得分 > 8 → 睡眠环境需优化
- 病史得分 > 8 → 长期问题

### 方案推荐逻辑

| 风险等级 | 推荐方案 |
|----------|----------|
| 低风险 | 预防保健方案 |
| 中等风险 | 日常护理方案 |
| 较高风险 | 日常护理方案 |
| 高风险 | 强化恢复方案 |
