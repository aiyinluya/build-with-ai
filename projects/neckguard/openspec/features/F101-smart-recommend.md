## 概述

根据用户的症状、历史记录和偏好，智能推荐最适合的运动和方案。

## 需求

### 推荐维度

1. **基于症状**
   - 根据当前症状推荐针对性运动
   - 颈椎痛 → 推荐颈部拉伸
   - 肩膀酸 → 推荐上斜方肌拉伸

2. **基于历史**
   - 分析用户喜欢的运动类型
   - 推荐相似运动

3. **基于时间**
   - 早上推荐激活类运动
   - 晚上推荐放松类运动

4. **基于进度**
   - 初学者推荐简单运动
   - 进阶者推荐困难运动

## 验收标准

- [ ] 根据症状推荐运动
- [ ] 根据时间推荐运动
- [ ] 推荐结果可解释
- [ ] 用户可反馈推荐质量
