## 概述

使用 Chart.js 将用户的症状评分数据可视化为折线图，直观展示健康趋势。

## 设计

### 图表类型

| 图表 | 用途 | 时间范围 |
|------|------|----------|
| 症状趋势折线图 | 展示症状变化 | 最近7/30/90天 |
| 运动打卡柱状图 | 展示运动频率 | 最近7/30天 |

### 数据展示

```javascript
// 折线图数据
{
  labels: ['3-18', '3-19', '3-20', '3-21', '3-22', '3-23', '3-24'],
  datasets: [
    {
      label: '颈椎不适',
      data: [5, 4, 6, 5, 4, 3, 4],
      color: '#ef4444',
    },
    {
      label: '肩膀酸痛',
      data: [4, 3, 5, 4, 3, 2, 3],
      color: '#f59e0b',
    },
    // ...
  ],
}
```

### 时间范围切换

- 最近7天
- 最近30天
- 最近90天
