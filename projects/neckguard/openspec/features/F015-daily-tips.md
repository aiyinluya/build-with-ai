# F015 - 每日健康提示

## 基本信息

| 属性 | 值 |
|------|-----|
| ID | F015 |
| 名称 | 每日健康提示 |
| 状态 | ✅ DONE |
| 优先级 | P3 |
| 创建 | 2026-03-25 |

## 概述

每天随机显示一条健康小贴士，提醒用户注意颈椎健康。

## 设计

### 提示内容

| 图标 | 标题 | 内容 |
|------|------|------|
| 🧘 | 定时休息 | 每工作45-90分钟，站起来活动5分钟 |
| 📱 | 正确用手机 | 手机举到视线高度，避免低头看 |
| 🛏️ | 侧睡注意 | 侧睡时手臂不要压在身下 |
| 💪 | 坚持运动 | 每天15分钟康复运动 |
| 🌡️ | 注意保暖 | 颈部保暖很重要 |
| 😴 | 优质睡眠 | 选对枕头 |
| 🖥️ | 显示器高度 | 屏幕顶端与视线平齐 |

### 显示规则

- 每天只显示一次
- 移动端显示在右下角
- 可手动关闭

## 验收标准

- [x] 提示随机显示
