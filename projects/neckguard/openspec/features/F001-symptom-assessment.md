# F001 - 症状自测问卷

| 属性 | 值 |

| ID | F001 |

## 概述

通过18道专业问题，从症状、工作习惯、睡眠、病史四个维度评估用户的颈椎健康状况。

## 设计

### 问卷结构

| 维度 | 题数 | 分类标签 |
|------|------|----------|
| 症状评估 | 7题 | 🩺 症状 |
| 工作习惯 | 5题 | 💻 工作 |
| 睡眠状况 | 4题 | 🛏️ 睡眠 |
| 病史记录 | 3题 | 📋 病史 |

### 评分机制

- 每题0-4分
- 总分72分
- 百分比换算

### 风险等级

| 百分比 | 等级 | 颜色 |
|--------|------|------|
| 0-20% | 低风险 | 🟢 绿色 |
| 21-45% | 中等风险 | 🟡 黄色 |
| 46-70% | 较高风险 | 🟠 橙色 |
| 71-100% | 高风险 | 🔴 红色 |
