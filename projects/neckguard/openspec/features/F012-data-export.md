## 概述

支持将用户的健康数据导出为 CSV 和 Markdown 格式，方便备份和分享。

## 设计

### 导出格式

| 格式 | 用途 | 内容 |
|------|------|------|
| CSV | 数据分析 | 症状记录表格 |
| Markdown | 阅读分享 | 健康报告文档 |

### CSV导出内容

```csv
日期,颈椎不适,肩膀酸痛,后背不适,睡眠质量,工作压力,备注
2026-03-24,5,4,3,6,7,今天工作较累
...
```

### Markdown导出内容

```markdown
# NeckGuard 健康报告

生成时间: 2026-03-24

## 症状记录

| 日期 | 颈椎 | 肩膀 | 后背 | 睡眠 | 压力 |
|------|------|------|------|------|------|
| ... | ... | ... | ... | ... | ... |

## 运动记录

- 2026-03-24: 收下巴训练
- ...
```
