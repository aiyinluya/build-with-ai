| 属性 | 值 |

| 项目名称 | NeckGuard |

| GitHub | github.com/aiyinluya/neckguard |

## 愿景

帮助伏案工作者了解和改善颈椎健康，通过科学的评估、个性化的方案和持续的追踪，让每个人都能拥有健康的颈椎。

## 核心价值

1. **科学专业** - 基于医学知识的评估和方案
2. **简单易用** - 直观的产品体验
3. **隐私优先** - 数据完全存储在本地
4. **开源透明** - 社区共建，持续改进

## 技术架构

### 技术栈

```
前端框架: React 18
构建工具: Vite 5
样式方案: TailwindCSS 3
图表库: Chart.js 4
路由: React Router 6
数据存储: LocalStorage
```

### 页面结构

```
/                    → 首页仪表盘
/assessment          → 症状自测
/exercise            → 康复方案/运动库
/track               → 追踪记录
/report              → 健康报告
/knowledge          → 知识库
/reminder            → 提醒设置
/profile             → 个人中心
```

## 开发规范

### 代码风格
- 组件使用函数式组件 + Hooks
- 命名遵循 React 惯例 (PascalCase)
- 样式使用 TailwindCSS 原子类
- 暗色模式使用 `dark:` 前缀

### Git提交规范
```
feat: 新功能
fix: Bug修复
docs: 文档更新
style: 样式调整
refactor: 重构
perf: 性能优化
test: 测试相关
```
